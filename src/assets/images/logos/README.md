# Logo Assets

Place your company logo files in this folder:

## Required Logo File:

- **re-company-logo.png** (or .jpg/.svg) - Main company logo for navigation

## Recommended Sizes:

- **Logo**: 200x60px or similar aspect ratio
- **Format**: PNG with transparent background preferred
- **File size**: Under 50KB for fast loading

## Usage:

The logo will automatically appear in the navigation bar replacing the fallback icon.
